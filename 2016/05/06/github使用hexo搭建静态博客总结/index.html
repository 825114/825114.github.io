##github使用hexo搭建静态博客总结

###1.准备
1）注册github,新建仓库，名称为xxxx.github.io。可以先配置好ssh免密钥登陆
2）安装node,https://nodejs.org/ 下载安装即可。
3）安装git for windows ,https://github.com/git-for-windows/git/releases/download/v2.8.2.windows.1/Git-2.8.2-64-bit.exe

###2.安装hexo
1)新建文件夹如F:\blog\



2）初始化hexo



hexo的安装至此完成。

###3.配置hexo
1)配置文件F:\blog_config.yml
vi打开，也可以用记事本之类的打开

修改_config.yml
title: 迷路的虾米 //博客标题
subtitle: 锅碗瓢盆，柴米油盐酱醋茶。//子标题
description: IT男的宅腐基地    //描述
author: TK    //作者，显示在左侧的
language: zh-cn //语言
timezone: Asia/Shanghai //时区
url: http://enjoythe.party //url
theme: yilia //主题设置，主题存在F:\blog\themes目录下
github配置，这里用ssh连接的，所以要先配置好SSH
deploy:
type: git
repository: git@github.com:username/username.github.io.git
branch: master
2)主题配置文件F:\blog\themes\yilia\_config.yml

###5.tips
1)设置文章摘要,文章不自动展开,要点more才展开?
文件需要截断处加上即可
2)修改主题
搜主题如concise主题，直接在F:\blog\下git clone

``` bash
git clone https://github.com/huangjunhui/concise.git themes/concise
``` 
修改配置文件F:\blog\_config.yml下的theme: concise
重新生成，部署
2)域名解析
CNAME解析
在F:\blog\source下新建CNAME文件，里面写你申请的域名，不带http://
然后域名管理后台那添加两条CNAME解析
记录类型     主机记录     解析线路     
``` bash
CNAME    www    默认    825114.github.io     
CNAME    @    默认    825114.github.io
```
