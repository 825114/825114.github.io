LPC54608在MDK-Keil5平台下开发环境搭建

  • LPC54608
NXP

有幸从NXP官网申请到一块LPC54608开发版，这是一块搭载4.3寸电容屏的开发版，
上电运行的touchGFX图形界面十分流畅细腻。可以从以下链接查看开发板的详细信息及视频展示。
OM13092: 面向LPC54608的LPCXpresso电路板

一、开发环境需要的安装包

1.1 首先需要安装驱动，双击[lpc_driver_setup.exe](https://community.nxp.com/servlet/JiveServlet/download/11529-389044-630660-377765/lpc_driver_setup.exe.zip)安装，
安装完成后，在设备管理器里能看到一个LPC based USB device,如下图：
![LPC based USB device](\image\nxp\lpc54608\0.png))

1.2 安装[LPCScrypt_installer_1.8.0_723 .exe](http://www.nxp.com/assets/downloads/data/en/software/LPCScrypt_installer_1.8.0_723.exe),这个是用来更新板载下载器固件用的，当板子背面JP1跳线连上后LPC-Link2处于DFU模式，装好LPCScrypt后，在开始菜单》所有程序里面可以找到LPCScrypt文件夹，里面有几个批处理文件。
在安装LPCScrypt是也会装上VCOM等的驱动。
